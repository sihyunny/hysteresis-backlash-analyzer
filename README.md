# Backlash Analyzer

A Python GUI application for analyzing hysteresis curves and measuring backlash in mechanical systems.

## Features

- **Backlash Measurement**: Automatic calculation of backlash from torque-angle data
- **Torsional Stiffness Analysis**: Calculate torsional stiffness for both positive and negative directions
- **Visual Output**: Generate high-resolution graphs with annotated measurements
- **Excel Logging**: Automatic logging of results to Excel file
- **Modern GUI**: CustomTkinter-based dark theme interface
